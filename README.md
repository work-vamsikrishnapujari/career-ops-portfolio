# Career Ops Portfolio

Personal portfolio website for Vamsikrishna, focused on Java/Spring Boot engineering, AWS, and Generative AI systems.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

This site deploys to GitHub Pages from the `main` branch using `.github/workflows/deploy.yml`.

Public URL:

```text
https://work-vamsikrishnapujari.github.io/career-portfolio/
```
